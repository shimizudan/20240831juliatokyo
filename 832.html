Julia のコード

  • カタラン数のリストからうどんの配置パターンを作成
  • 逆に配置パターンからカタラン数のリストを決定する

soi関数

  • うどんを合成する関数です。
soi([1], [2]) = [1, 2]
soi([1], [2, 3]) = [1, 2, 3]
soi([1, 2], [3]) = [2, 1, 3]
soi([1, 2, 3], [4]) = [3, 2, 1, 4]

カタラン数のリストチェック

  • 1と2が同数あるリストを用いて，左端からどの番まででも（1の個数）≧（2の個数）となっているかチェックします。
catalan_check([1, 1, 1, 1, 2, 2, 2, 2]) = true
catalan_check([1, 2, 1, 1, 2, 1, 2, 2]) = true
catalan_check([2, 2, 2, 2, 1, 1, 1, 1]) = false
catalan_check([1, 2, 2, 1, 2, 1, 1, 2]) = false

soi_seqの逆関数

  • 結局，アルゴリズムはよくわからなかったです。
  • 関数は１対１だったので，ペアのリストを作ってチェックしました。
  • １対1は明らかではなかったのですが，一応確認は取れています。
inverse_soi_seq([1, 2, 4, 3, 5, 6]) = [1, 2, 1, 2, 1, 1, 2, 2, 1, 2]
inverse_soi_seq([1, 2, 4, 3, 6, 5]) = "対応するカタラン数のリストはない"
inverse_soi_seq([1, 2, 3, 4, 6, 5]) = "対応するカタラン数のリストはない"
